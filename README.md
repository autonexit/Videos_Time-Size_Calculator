# Videos_Time-Size_Calculator
این نرم‌افزار یک ابزار ساده و کاربردی برای محاسبه مجموع زمان و حجم فایل‌های ویدیویی داخل یک پوشه است. با انتخاب فولدر، تمام ویدیوها اسکن می‌شوند، مدت زمان هر فایل محاسبه شده و در نهایت مجموع ساعات پخش و حجم کل به‌صورت دقیق نمایش داده می‌شود. مناسب برای آرشیو، مدیریت محتوا و تولیدکنندگان ویدیو
